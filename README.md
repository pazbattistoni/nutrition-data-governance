NutriGovernance ETL Pipeline

Este proyecto implementa un pipeline de ETL (Extracción, Transformación y Carga) diseñado bajo principios de Gobernanza de Datos. El sistema automatiza la ingesta de información nutricional desde APIs externas, aplicando procesos rigurosos de limpieza, normalización y validación de integridad.

Objetivos principales:
Calidad de Datos: Implementación de reglas de negocio para asegurar la completitud y consistencia de macronutrientes.
Estandarización: Unificación de métricas y unidades para garantizar la comparabilidad de los activos de datos.
Arquitectura Escalable: Uso de contenedores para asegurar la portabilidad del entorno y un flujo de datos desacoplado.

  Stack Técnico: Python (Pandas/Requests), PostgreSQL, Docker.
